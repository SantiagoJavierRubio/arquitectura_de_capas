# Deploy en Heroku
Desafío entregable para el curso de backend de Coderhouse

## Para probar el código:

```bash
# Clonar el repositorio
$ git clone https://github.com/cabrerarodrigo/heroku-deploy.git

# Instalar dependencias
$ npm install

# Iniciar el servidor
$ npm start
```
